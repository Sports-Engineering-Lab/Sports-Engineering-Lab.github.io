# Media Coverage

## Format
각 미디어 항목은 다음 형식을 따릅니다:
```yaml
- date: YYYY-MM-DD
  title: 제목
  thumbnail: thumbnails/image_name.jpg
  link: https://example.com
  description: 간단한 설명 (선택사항)
```
